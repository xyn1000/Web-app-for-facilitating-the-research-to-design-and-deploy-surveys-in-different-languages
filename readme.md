# Backend Server

## Deploy & Usage

upload backend.zip to cloud server

unzip backend.zip

```bash
cd cp13-backend
pip3 install -r requirements.txt
python3 manage.py runserver 0.0.0.0:8000
```

# Frontend Server

## Deploy & Usage

upload frontend.zip to cloud server

unzip frontend.zip

```bash
npm install -g serve
serve -s dist
```
